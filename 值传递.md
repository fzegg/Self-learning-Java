# 方法的形参的传递机制：值传递机制

- ### 形参：方法定义时，声明的小括号内的参数

  ### 实参：方法调用时，实际传递给形参的数据

- ### 值传递机制：如果参数时基本数据类型，此时实参赋给形参的是实参的实际值

  ###                        如果参数是引用数据类型，此时实参赋给形参的是实参真实存储的地址值

```java
//并不能完成交换，交换的是swap方法中的m和n（方法在调用之后即弹出栈）
public static void main（String[] args）{
	int m =	10;
    int n = 20;
    test.swap(m, n);
}	
public void swap(int m, int n){
    	int temp = m;
    	m = n;
    	n = temp;
}
```

```java
public class Data{
    int m;
    int n;
}
public static void main（String[] args）{
    Data data = new Data();
    data.m = 10;
    data.n = 20;
    test.swap(data);
}
public void swap(Data data){
    int temp = data.m;
    data.m = data.n;
    data.n = temp;
}
```
